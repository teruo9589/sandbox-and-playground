## 3. 画面項目定義

### 3.1 項目IDプレフィックス規則

部品種別に応じて以下のプレフィックスを使用し、`プレフィックス_名称` の形式で命名する（例: `input_name`, `lbl_created_at`）。一覧に該当する部品種別がない場合は、最も近いプレフィックスを使用するか、プロジェクト内で新規プレフィックスを定義してこの表に追記する。

**入力系**

| プレフィックス | 対象部品種別 | 用途・特記事項 |
|---|---|---|
| `input_` | テキストボックス | 1行テキストを入力する最も基本的なフィールド |
| `pwd_` | パスワード入力 | マスク表示されるテキスト入力 |
| `num_` | 数値入力 | HTMLの `type="number"` 相当。数値以外を入力させたくない場合に使用 |
| `textarea_` | テキストエリア | 複数行テキストを入力。改行可能 |
| `select_` | セレクトボックス（単一選択） | ドロップダウンを開いて1件選択 |
| `mselect_` | セレクトボックス（複数選択） | ドロップダウンを開いて複数件選択 |
| `combo_` | コンボボックス（単一選択） | 入力によるフィルタ＋1件選択 |
| `mcombo_` | コンボボックス（複数選択） | 入力によるフィルタ＋複数件選択 |
| `chk_` | チェックボックス | 選択肢を一覧で並べて表示する複数選択 |
| `rdo_` | ラジオボタン | 選択肢を一覧で並べて表示する単一選択 |
| `date_` | 日付ピッカー | カレンダーUIで日付を選択 |
| `time_` | 時刻ピッカー | 時刻入力UIで時刻を選択 |
| `datetime_` | 日時ピッカー | カレンダーUIと時刻ピッカーで日付・時刻を同時選択 |
| `file_` | ファイルアップロード | ファイルを選択してアップロード。単一・複数ファイル対応 |
| `tgl_` | トグル／スイッチ | クリックでON/OFFを切り替える2値入力 |
| `rng_` | スライダー | スライダーを操作して数値や範囲を選択 |
| `color_` | カラーピッカー | カラーパレットやスポイトで色を選択 |
| `rating_` | レーティング | 星などのアイコンをクリックして評価値を入力 |
| `editor_` | リッチテキストエディタ | WYSIWYG形式のテキスト入力 |
| `treeselect_` | ツリービュー（選択操作あり） | 階層構造のデータからノードを選択 |

**表示系**

| プレフィックス | 対象部品種別 | 用途・特記事項 |
|---|---|---|
| `lbl_` | ラベル | テキスト表示専用 |
| `img_` | 画像 | 静的・動的画像の表示 |
| `lnk_` | リンク | クリックで画面遷移または外部URLへ誘導するテキストリンク |
| `tbl_` | テーブル | 一覧・グリッド表示。テーブル内に入力項目を含む場合は、各入力項目を個別のプレフィックスで定義する |
| `badge_` | バッジ | ステータス・件数などの強調表示 |
| `icon_` | アイコン | ラベルと独立して識別が必要なアイコン。クリックなどの操作を伴う場合は `btn_` を使用 |
| `prog_` | プログレスバー | 処理の進捗割合をバーで表示 |
| `chip_` | チップ／タグ | 選択済み項目などの強調表示 |
| `avatar_` | アバター | ユーザーアイコン・プロフィール画像 |
| `chart_` | チャート | グラフ・チャート（棒・折れ線・円など） |
| `tooltip_` | ツールチップ | ホバー時に表示、操作不可 |
| `tree_` | ツリービュー（表示専用） | 階層構造のデータ表示のみ |

**レイアウト・ナビゲーション系**

| プレフィックス | 対象部品種別 | 用途・特記事項 |
|---|---|---|

| `toast_` | トースト通知 | 処理結果などの一時表示 |
| `stepper_` | ステッパー | ウィザード形式の手順進捗表示 |
| `drawer_` | ドロワー | サイドから出るスライドパネル |
| `popover_` | ポップオーバー | クリック時に表示、操作可能（ボタン・リンクを含められる） |
| `pgn_` | ページネーション | 一覧のページ切替 |
| `spinner_` | ローディングスピナー | 処理中の待機表示 |

### 3.2 画面項目一覧

- 入力・表示すべての項目を記載します。
- **種別**: `入力` / `表示` で区別。
- **部品種別**: 3.1 項目IDプレフィックス規則の「対象部品種別」を参照。
- **データ型**: `文字列` / `数値` / `日付` / `日時` / `真偽値` など。
- **文字種**: `半角` / `全角` / `半角英数` / `半角数字` / `-`（制限なし・表示専用）など。
- **表示形式**: 日付・数値・コードなど画面上のフォーマット。制限なしの場合は `-`。
- **必須**: `✓`: 必須 / `-`: 任意。
- **初期値**: 画面表示時のデフォルト値。なしの場合は `-`。
- **最大長**: 入力可能な最大文字数。制限なし・表示専用の場合は `-`。
- **DBテーブル / DBカラム**: DBと紐づかない集計値・計算値は `-`。
- **ロール列**: `表示/活性` の順で記載（例: `○/○`）。活性制御が不要な要素（ラベルなど）は `○/-` / `×/-`。
- **ロール凡例**: `○`: 該当 / `×`: 非該当 / `-`: 制御なし

| No. | 項目名 | 項目ID | 種別 | 部品種別 | データ型 | 文字種 | 表示形式 | 必須 | 初期値 | 最大長 | DBテーブル | DBカラム | 一般ユーザー(表示/活性) | 管理者(表示/活性) | ゲスト(表示/活性) | その他(表示/活性) | 備考 |
|-----|--------|--------|:----:|---------|---------|-------|---------|:----:|--------|--------|-----------|---------|:----------:|:-----:|:-----:|:-----:|------|
| 1 | 〇〇名 | `input_name` | 入力 | テキストボックス | 文字列 | 全角 | - | ✓ | - | 50文字 | `items` | `name` | ○/○ | ○/○ | ×/- | ○/× | |
| 2 | 種別 | `select_type` | 入力 | セレクトボックス（単一選択） | 文字列 | - | - | ✓ | 未選択 | - | `items` | `type` | ○/○ | ○/○ | ×/- | ○/× | マスタ参照 |
| 3 | メモ | `textarea_memo` | 入力 | テキストエリア | 文字列 | - | - | - | - | 500文字 | `items` | `memo` | ○/○ | ○/○ | ×/- | ○/× | |
| 4 | 登録日時 | `lbl_created_at` | 表示 | ラベル | 日時 | - | `YYYY/MM/DD HH:mm` | - | - | - | `items` | `created_at` | ○/- | ○/- | ×/- | ○/- | |
| 5 | ステータス | `lbl_status` | 表示 | ラベル | 文字列 | - | `0:無効 / 1:有効` | - | - | - | `items` | `status` | ○/- | ○/- | ×/- | ○/- | |
| 6 | 件数 | `lbl_total_count` | 表示 | ラベル | 数値 | - | `#,##0` | - | - | - | - | - | ○/- | ○/- | ×/- | ○/- | 集計値 |

## 4. ボタン・アクション定義

### 4.1 ボタンIDプレフィックス規則

すべてのボタンIDは `btn_` を使用し、`btn_名称` の形式で命名する（例: `btn_search`, `btn_delete`）。

### 4.2 ボタン・アクション一覧

- **連携パラメータ**: サーバーサイドへ送信するパラメータを記載。項目IDで参照し、固定値は値を直接記載。不要な場合は `-`。
- **ロール列**: `表示/活性` の順で記載（例: `○/○`）。
- **ロール凡例**: `○`: 該当 / `×`: 非該当 / `-`: 制御なし

| No. | ボタン名 | ボタンID | アクション | 連携パラメータ | 遷移先 | 一般ユーザー(表示/活性) | 管理者(表示/活性) | ゲスト(表示/活性) | その他(表示/活性) | 備考 |
|-----|---------|---------|-----------|--------------|--------|:----------:|:-----:|:-----:|:-----:|------|
| 1 | 検索 | `btn_search` | 検索APIを呼び出し一覧を更新 | `select_type`, `input_name` | 同画面 | ○/○ | ○/○ | ○/× | ○/○ | |
| 2 | 新規登録 | `btn_new` | 登録画面へ遷移 | - | SCR-XXX | ○/○ | ○/○ | ×/- | ○/× | |
| 3 | 登録実行 | `btn_register_exec` | 入力内容を登録APIへ送信 | 3.2 画面項目一覧の入力項目すべて | SCR-XXX | ○/○ | ○/○ | ×/- | ○/× | 登録画面に配置 |
